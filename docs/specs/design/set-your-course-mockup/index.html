Set Your Course — Static Scenarios

Fifteen scenarios of the new Set Your Course screen, rendered in Brightpath tokens. Everything below is static — no backend, no Gemma, no live animation beyond CSS keyframes. Scroll, screenshot the ones worth keeping, then make the three synthesis calls in the box below.

Decisions to make

  1. Community Suggestions visibility threshold. Render the section at count ≥ 1 (aggressive, shows the mechanism early) or count ≥ 3 (safer, nothing social until a crowd actually exists)? See scenarios 10, 11, 12. The n=1 case is the PM's "surveillance with n=1" concern made visible.
  2. Middle chip label. Ship "Show me less common paths" (spec default, taxonomic) or "Show me weirder jobs" (PM critique, kid-voiced)? See scenario 5. Rendered side-by-side; the third-chip ("Wrong major") is fixed.
  3. School Discovery in v0.5. Stub the tile and ship the dead-link destination (scenario 13), cut the tile entirely, or build a minimal Discover screen? See scenario 13. Stub is the fastest path; the tile is already in the design, the /discover route can be a "coming soon" one-liner.

05 · Chip label decision · A vs B

Middle chip — "Show me less common paths" vs "Show me weirder jobs"

Same chip rail, same weights, different label on the middle chip. Variant A is the spec default (taxonomic but clear). Variant B is the PM's kid-voiced rewrite. The third chip is fixed: "Wrong major".

Decision 2: Which reads like a student, which reads like a site?
Desktop · side-by-side variants
Variant A · Spec default

Reads as product copy. Honest about what the toggle does (reveals stretch tiers). Matches the rest of the spec's voice-of-the-tool language.

Variant B · PM critique · kid-voiced

Reads as the student. Matches the founder's own "I don't see my shit" quote. Risk: "weirder" slightly over-promises — the tier is unusual, not literally weird.
