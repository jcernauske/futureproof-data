# Gemma 3n Impact Challenge — Winner Videos & Writeups

**Purpose:** Study these before scripting the FutureProof video. Video is 30% of judging.

## Key Takeaway

Every winner had a deeply personal story. First place built for his blind brother. Third place built for Eva, a graphic designer with cerebral palsy. Dream Assistant built for a specific user with speech impairments. LENTERA served disconnected rural communities. **The video is a story about a person, not a product demo.**
